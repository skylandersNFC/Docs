All thx to "ItsSecretNate" and "Maffama'iks" (@maff), who told me about the process.

You can generate the "Heartbreaker Buckshot sky" dump with the RPCS3 emulator for PS3.

**Heartbreaker Buckshot IDs**
`ID 618   VARIANT_ID 21516`

I'm not sharing it as file, since people will start thinking they can flash it on NFC cards.
This is not possible, it only works inside an emulator with the "workaround" i'm showing.
